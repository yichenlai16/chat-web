# Chat-Web

WebSocket Chat App using Vue 3 with Pinia, Express

![demo gif](https://i.imgur.com/F5wc0Od.gif)


## Quick Start

### Client

```bash
cd client
npm i
npm run dev
```
### Server

```bash
cd server
npm i
npm run dev
```

After the setup, you can access the application at:

**URL:**
- http://localhost:5173
